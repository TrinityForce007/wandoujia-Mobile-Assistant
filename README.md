# 豌豆荚手机助网站

> A Vue.js project

## 安装使用

``` bash
# 安装依赖
npm install

# 连接数据库服务，监听3000端口
cd src/server
node index

# 打包
cnpm run dev

# build for production and view the bundle analyzer report
npm run build --report
```
